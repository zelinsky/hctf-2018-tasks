PWN Coin ICO

Wait! I know what you're thinking: "Yet another standard ICO". Wrong!

The PWN University's library team has created a new crypto currency from scratch, redefining how blockchain can be used. We implemented an entirely new way of storing, managing and spending crypto currencies.

So, how does it work? Much like our biggest influence IOTA, PWN Coins are build on two principles:

De-Decentralization

The main caveat of blockchain is the consensus, with complex algorithms that require proofs of work. We narrowed the problem down to decentralization. Therefore we implemented the first fully de-decentralized blockchain, using a central trust to make fast decisions.

Using Keyless Cryptography

Since cryptography adds a lot of complexity to every implementation and is often the single point of failure for a system we decided to use a keyless approach. Implementing our own keyless crypto system proofed to be much more efficient than any other system.
